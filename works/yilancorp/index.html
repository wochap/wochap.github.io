Resumen

Gracias a la experiencia con anteriores proyectos, fue facil desarrollar el sitio, haciendo uso de mi framework personal wocss (inspirado en InuitCSS) y Jquery, para las animaciones use VelocityJS, tambien hize uso de Scrollspy (modulo de MaterializeCSS), tuve que modificarlo para que funcionara con un diferente scroll container.

Tecnologías destacadas:

  • Service worker
  • CSS Animations
  • JSONP
  • CSS FlexBox

Workflow:

  • CSS: SASS, BEMIT, ITCSS, WOCSS.
  • JS: Jquery, ES2015, ESLint.
  • Build System: Webpack + Babel + custom-loaders.

Logros:

  • Personalizar el plugin Scrollspy de MaterializeCSS, para que funcionase con un diferente scroll container.
  • Fancy animations.
