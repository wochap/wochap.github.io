Summary

Thanks to the experience gained with previous projects, it was easy to develop the site, using my personal framework wocss (inspired by InuitCSS) and Jquery. For animations i used VelocityJS, i also used Scrollspy (MaterializeCSS module), i had to modify it to work with a different scroll container.

Featured tech:

  • Service worker
  • CSS Animations
  • JSONP
  • CSS FlexBox

Workflow:

  • CSS: SASS, BEMIT, ITCSS, WOCSS.
  • JS: Jquery, ES2015, ESLint.
  • Build System: Webpack + Babel + custom-loaders.

Achievements:

  • Customize the Scrollspy plugin of MaterializeCSS, to work with a different scroll container.
  • Fancy animations.
